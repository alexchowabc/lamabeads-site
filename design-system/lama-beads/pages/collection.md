# Collection Page Override

## Page Job

Collection is the browsing and comparison page. It should help users scan many products without feeling like a landing page.

## Structure

1. Compact page heading.
2. Search and filters.
3. Product grid.
4. Empty state when no product matches.

## CTA Rules

- Product cards are the primary interaction.
- Avoid repeated global contact buttons above the grid.
- Contact CTA can appear after a user opens a product or reaches the page footer.

## Media Rules

- Use one preview image per product card.
- Keep card metadata compact: name, category, availability or material cue.
- When there are many products, prioritize lazy loading and stable aspect ratios.
