# Product Detail Page Override

## Page Job

Product Detail is the inspection page. It should support photos, future videos, descriptions, materials, origin, meaning, and care without duplicating media.

## Structure

1. Main media viewer with thumbnail rail.
2. Product name, category, description, and key specs.
3. One primary contact CTA.
4. Optional media library only when there are videos or additional non-duplicated assets.
5. Related products.

## CTA Rules

- One primary CTA: contact for consultation.
- Secondary actions should be text links, not pill buttons.
- Avoid repeating "view photos" when thumbnails already sit next to the main image.

## Media Rules

- Do not show the same photo in both the thumbnail rail and a lower photo strip.
- Future videos should appear as video thumbnails in the same media system or a dedicated video section.
- Keep image and video containers dimensionally stable to prevent layout shift.
